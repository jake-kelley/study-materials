Case Study Technique

  • Tabs: Overview / Business Req / Technical Req / Existing Env + 6–8 Q.
  • Read the question first, then hunt constraint keywords: RTO, RPO, cost, region/sovereignty, compliance, latency, HA%.
  • May not be able to return once you leave the block — don't rush.
  • Design exam: many answers "work"; constraints pick the best.

Identity — Entra ID

  • Tiers Free / P1 (Conditional Access, dynamic groups, SSPR writeback) / P2 (+PIM, Identity Protection, Access Reviews).
  • Hybrid sync: PHS / PTA / Federation(ADFS).
  • RBAC = authorize resource access; Conditional Access = context sign-in; B2B guest, B2C customer.

Governance

  • Hierarchy: Mgmt Group → Subscription → Resource Group → Resource (inherit down).
  • Azure Policy = what's allowed (Deny/Audit/Append/DINE); RBAC = who can do what.
  • Locks (CanNotDelete/ReadOnly), tags, Blueprints.
  • Key Vault = secrets/keys/certs (Std vs Premium/HSM) + managed identities.

Monitoring

  • Azure Monitor umbrella; Log Analytics (KQL); App Insights (APM).
  • Diagnostic settings → Log Analytics / Storage(archive) / Event Hubs(SIEM).
  • Alerts + Action Groups; Sentinel = SIEM/SOAR.

vCore tiers: General Purpose / Business Critical (local SSD+HA replica) / Hyperscale (100TB). Serverless=auto-pause.

Cosmos DB

  • APIs: NoSQL(Core), MongoDB, Cassandra, Gremlin(graph), Table. Match existing app.
  • Consistency (strong→weak): Strong → Bounded Staleness → Session(default) → Consistent Prefix → Eventual.
  • Multi-region writes, global, <10ms.

Business Continuity

  • Azure Backup = restorable copies (RPO). Site Recovery (ASR) = replicate + region failover (RTO).
  • HA: Availability Set (fault/update domains, 1 DC, 99.95%) vs Zone (separate DCs, 99.99%) vs region pairs.
  • VM Scale Sets = autoscale identical VMs.

Migration + Frameworks

  • Azure Migrate hub; DMS for DBs; Data Box offline bulk.
  • CAF: Strategy→Plan→Ready→Adopt→Govern→Manage. "R"s: rehost/refactor/rearchitect/rebuild.
  • WAF pillars: Reliability, Security, Cost, Operational Excellence, Performance.

Top Traps

Backup(RPO) vs ASR(RTO) · SQL DB vs MI(lift-shift) vs VM · LB/AppGw/FrontDoor/TrafficMgr (L4/L7×regional/global) · VPN vs ExpressRoute · P1 vs P2(PIM) · Set 99.95 vs Zone 99.99 · LRS<ZRS<GRS<GZRS · Cosmos consistency order
